写作技巧:

1. 清晰、聚焦、相关、明确、避免模糊的词语
2. 用`###`或`"""`将指令和待处理的内容分开
3. 指定输出的格式
4. 角色扮演
5. 提供样本
6. 持续多轮对话（英文环境）

四大要素: 指令词、背景信息、输入文本和输出要求 

- 适当的提示引导模型生成有用的输出
- 多种方式表达
- 描述具体的事件附加相关的背景信息
